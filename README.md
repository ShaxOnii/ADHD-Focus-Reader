# ADHD-Focus-Reader
🇬🇧 EN: A smart Chrome extension designed for people with ADHD. It uses Bionic Reading principles to bold words and plays context-aware ambient music to improve focus.  🇵🇱 PL: Inteligentna wtyczka do Chrome dla osób z ADHD. Ułatwia czytanie dzięki technice pogrubiania liter (Bionic Reading) i odtwarza muzykę tła dopasowaną do czytanego tekstu.
